## Overview

Greek & Latin Bold Unicode Converter is a small browser tool for turning Greek, Latin, and numeric text into bold mathematical Unicode characters. It supports sans-serif bold and serif bold output, optional Greek accent stripping, uppercase conversion, sample text, clearing, and clipboard copy.

The tool is packaged as a self-contained static page so it can run directly from the portfolio without extra dependencies.
