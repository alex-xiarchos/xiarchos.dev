I'm Alexandros Xiarchos, a web developer with a Computer Engineering background
and a focus on building practical, maintainable software. My work spans
frontend development, data-focused systems, information retrieval, and low-code
applications.

I enjoy projects where the interface and the underlying system need to work
cleanly together: responsive React apps, PHP and JavaScript platforms, search
and ranking pipelines, spatial data structures, and cloud-deployed business
tools.
